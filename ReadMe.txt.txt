About Bingo App:

1. BingoCard class
It represents the Bingo card grid which can be Bingo won!

2. Bingo class
It has solution for part 1 and part 2 as per the question given

3. Main class
It has example of passing the input and get the result

4. BingoTest class
It represents the Junit for Bingo class

Prerequisite:
1. JDK 17+
2. Gradle
3. Intellij Idea

To Run the application:
1. Open the project in Intellij.
2. Open main class and run it.
3. The output will be displayed in console.
